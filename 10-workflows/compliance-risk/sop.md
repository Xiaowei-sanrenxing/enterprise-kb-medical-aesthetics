# 合规与风控 SOP · 人可读总览 ★医美核心

> 给人看的全流程。Agent 执行请走 [`_task-card.md`](_task-card.md)。
> 医美第一权重：合规前置，高风险动作人工把关，纠纷分级处理。

## 一句话流程
先分流（上新发布 / 客诉纠纷）→ 上新先核资质再过广告合规、经审查留痕才发 → 客诉先定级、agent 只处理最轻级、其余必转人工 → 隐私全程🔴受限脱敏 → 高风险动作一律人工+法务/医师。

## 铁律（本工作流最严，逐条不可破）
1. **资质可追溯，超范围零容忍**：机构《医疗机构执业许可证》、医师资格证+执业证+执业范围、药械正规来源齐全且对得上，缺一或超范围 → 🛑 停，不得发布/上新。
2. **医疗广告不越线**：禁疗效保证、禁治愈率/有效率、禁绝对化用语（最/第一/100%）、禁前后对比图诱导、禁用患者形象作证明、须经审查、严格区分"医疗"与"生活美容"。
3. **隐私永不外发**：术前术后照片/病历/诊疗记录/客户个人信息 = 医疗隐私🔴受限，绝不进定版、绝不对外、对外输出必脱敏打码，授权使用须书面同意。
4. **纠纷分级，agent 只碰最轻一级**：仅"轻度不满意"可由 agent 安抚+登记+上报；效果争议/疑似不良反应/舆情/诉讼 → 立即人工。
5. **高风险动作 agent 绝不决策**：赔付、对外声明、舆情回应、超范围放行、医疗广告放行 → 强制人工+法务/医师把关。
6. **不确定就停**：任何拿不准 → 默认拦截并上报〔合规负责人〕，宁可拦错不可放过。
