# 资质核验与医疗广告合规清单（医美核心）★

> 核心：**发布对外内容 / 上新项目前，先过这两道闸门。任一项勾不掉 → 🛑 停，转〔合规负责人〕，不得放行。**
> 法规以《医疗广告管理办法》《广告法》及主管部门要求为准（此处讲要点，具体条款以官方文本为准）。

## 一、资质核验清单（上新 / 上项目前，agent 逐项核对，缺一即拦）

| 核验项 | 要点 | 不达标动作 |
|---|---|---|
| ☐ 机构资质 | 〔机构名〕持有效《医疗机构执业许可证》，诊疗科目覆盖该项目 | 🛑 停，转〔合规负责人〕 |
| ☐ 医师资质 | 操作医师持《医师资格证》+《医师执业证》，且**执业范围**覆盖该项目 | 🛑 停，超范围零容忍 |
| ☐ 药械来源 | 注射类/光电类药械经正规渠道采购、有合法批准文号、可溯源 | 🛑 停，禁用来源不明药械 |
| ☐ 无超范围操作 | 项目不超出机构诊疗科目与医师执业范围 | 🛑 超范围零容忍，绝不放行 |

> 证照原件 = ⛔禁止入库（证件类，留原地索引）；agent 只读核验结论，不留存证件影像。

## 二、医疗广告合规自检清单（对外内容发布前，逐条勾选，任一勾不掉 → 改稿或拦截）

- ☐ 无**疗效保证**（不写"保证有效/必定见效/无效退款"等承诺）
- ☐ 无**治愈率 / 有效率 / 成功率**等数字承诺
- ☐ 无**绝对化用语**（最/第一/顶级/100%/唯一/根治）
- ☐ 无**前后对比图**诱导（术前 vs 术后对比图禁用）
- ☐ 不**利用患者形象 / 名义 / 评价作证明**（禁患者现身说法、禁好评截图当广告）
- ☐ 已**送医疗广告审查**并取得审查通过（医疗广告须先审后发）
- ☐ 严格区分"**医疗美容**"与"**生活美容**"，不把医疗项目包装成生活美容规避监管
- ☐ 无夸大、无误导、无与同行作贬低性比较

### 红线表（命中即拦截，无例外）
| 红线 | 典型违规话术（示例，禁用） |
|---|---|
| 疗效保证 | "包好""无效退款""一次见效永久不反弹" |
| 数字承诺 | "有效率 99%""治愈率行业第一" |
| 绝对化 | "全市最好""第一品牌""100% 安全无风险" |
| 前后对比 | 配"术前→术后"对比图诱导下单 |
| 患者证言 | "顾客实拍好评""真人现身说法" |
