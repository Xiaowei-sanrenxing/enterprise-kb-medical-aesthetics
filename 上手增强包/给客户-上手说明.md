# 📦 企业知识库 · 通用上手增强包

> 这个包是配合你那套「企业 AI 知识库」一起用的**上手辅助工具**。
> 目的只有一个：**让你不用纠结"怎么跟 AI 开口"，照着做就能用顺。**

---

## 这个包里有什么

| 文件 | 给谁 | 干啥 |
|---|---|---|
| **一键安装-skill.command** | 用 Claude Code 的人 | 双击，给你装两个"自动化按钮"（喂料 + 体检）|
| **skills/** | （安装脚本用的）| 两个技能的源文件，不用手动碰 |

> 📌 **话术速查卡在哪？** 在你那套知识库的**根目录**里（`话术速查卡.md`），而且已经**按你的行业定制好了**（例子套的是你本行业的真实工作流）。所以这里不再放一份，避免两头打架。

---

## 你属于哪种用法？（二选一）

### 🅰️ 你用飞书 / 网页版 AI（不懂技术，最常见）

**什么都不用装。** 直接打开你知识库根目录的 **《话术速查卡.md》**，照着上面的句子跟 AI 说话就行。

> 那两个技能（skill）是 Claude Code 专用的，你这种用法装了也调不了——**但完全不影响**，话术卡里每个动作都给了"说人话"的版本，效果一样。

---

### 🅱️ 你自己用 Claude Code（技术型）

多一个福利：可以装两个**斜杠快捷键**，以后维护库连话都不用编。

**第 1 步 · 安装**
双击 **`一键安装-skill.command`**（Mac）。它会把两个技能装进你的 Claude Code。

> 装不了 / 不是 Mac？手动也行：把 `skills/` 里的 `company-brain-ingest` 和 `company-brain-audit` 两个文件夹，拷到你电脑的 `~/.claude/skills/` 目录下即可。

**第 2 步 · 确认装好**
开个新的 Claude Code 会话，输入 `/company-brain`，能看到 ingest / audit 跳出来就成了。

**第 3 步 · 以后这么用**（敲斜杠点名，不用组织语言）
- 喂新料入库 → 输入 `/company-brain-ingest`，把文件一丢
- 给库做体检 → 输入 `/company-brain-audit`
- 其余日常用法 → 看知识库根目录的 **《话术速查卡.md》**

---

## 一句话放心丸

> 这两个技能只是**加速按钮**，不是命根子。**装不装、用不用，你的知识库都能照常运转**——所有规矩都已经写进库里的 `90-governance/` 文档和根目录《话术速查卡.md》了。我们不用任何插件绑架你。
