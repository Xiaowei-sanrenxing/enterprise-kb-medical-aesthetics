# 任务卡 · 获客引流（traffic-acquisition）· 脱敏示范

> Agent 操作总台。接到"写种草内容/平台投放/设计到店钩子/团购上架"类任务,从这里开始。

## 🎯 目标(Goal)
用合规内容把目标客群从公域(小红书/抖音/大众点评)拉到私域或到店,产出"高意向、非白嫖"的线索,顺畅交接到面诊转化,全程不踩医疗广告红线。

## 📥 上下文(Context) —— 动手前必读全
1. [`context/医美三平台种草矩阵.md`](context/医美三平台种草矩阵.md) — 三平台分工 + 内容形态 + 合规注意
2. [`context/到店钩子设计与转化路径.md`](context/到店钩子设计与转化路径.md) — 钩子怎么设、怎么避白嫖、怎么接面诊
3. 〔机构资质/项目清单〕 — 可对外宣传的项目与医师资质(真实库含此项)
4. 〔投放数据/ROI 看板〕 — 各平台获客成本与核销率(真实库含此项)

## 🪜 指令(Instructions)
1. 定目标平台 → 按三平台矩阵选内容形态
2. 写内容 → 过合规自检(禁疗效/禁前后对比图诱导/禁绝对化/标资质)
3. 选钩子(引流款/团购券/到店礼)→ 设防白嫖门槛
4. 设引流路径 → 公域钩子 → 私域承接 or 到店核销
5. 上架团购/发布内容 → 埋追踪标识
6. 到店/留资客户 → 交接 [[在线咨询到到店转化SOP]]
7. 回流数据 → 记录各钩子的到店率/白嫖率

## 🛠 工具(Tools)
〔内容排期/发布工具〕 · 合规话术库检索 · 〔团购后台/核销系统〕 · 〔获客数据看板〕

## 🔒 权限(Permission)
- 可读:合规话术、项目清单、投放数据、钩子设计
- 🔴 受限:客户术前术后照片/病历/姓名手机微信 → 绝不用于对外内容、输出必脱敏打码
- ⛔ 禁止:对外内容用前后对比图诱导、宣称疗效保证、绝对化用语;涉及项目须可追溯医师资质
- 需人工确认:超出常规的引流价、新项目对外文案上线、跨平台大额投放

## ✅ 验收(Acceptance)
- 对外内容零疗效话术 ｜ 零前后对比图诱导 ｜ 零绝对化用语 ｜ 已标医疗机构资质
- 钩子设了防白嫖门槛 ｜ 引流路径能接面诊 ｜ 隐私零外发
- 详见 evals.md

# 获客引流 SOP · 人可读总览

> 给人看的全流程。Agent 执行请走 [`_task-card.md`](_task-card.md)。

## 一句话流程
定平台 → 按三平台矩阵写合规内容 → 过红线自检 → 选引流钩子设防白嫖门槛 → 发布/上架团购 → 私域承接或到店核销 → 交接面诊转化 → 回流全链路数据。

## 几条铁律
1. **三平台分工不混用**：小红书种草拦截搜索、抖音放量+本地团购、大众点评临门决策，各司其职。
2. **合规高于流量**：宁可少曝光，绝不碰疗效保证、前后对比图诱导、绝对化用语；项目须可追溯医师资质。
3. **钩子要防白嫖**：低价引流款必须设门槛（到店面诊/限新客/绑定升单路径），不做纯薅羊毛入口。
4. **引流必接面诊**：所有钩子的终点是到店面诊，不是核销完就断，留资即转 [[在线咨询到到店转化SOP]]。
5. **隐私红线**：客户术前术后照片/病历/个人信息 🔴 受限，绝不进对外内容，输出必脱敏。

## 详细资产
- [`context/医美三平台种草矩阵.md`](context/医美三平台种草矩阵.md) — 小红书/抖音/大众点评分工 + 内容形态 + 合规注意
- [`context/到店钩子设计与转化路径.md`](context/到店钩子设计与转化路径.md) — 引流款/团购/到店礼 + 防白嫖 + 接面诊
- 〔机构资质/项目清单〕 — 客户补：可对外宣传的项目与医师资质

## 动作题
4. 写一条小红书医美种草，发布前必做的三步合规自检是什么？（答：去疗效话术、去前后对比图诱导、加医疗机构资质标注）

# 到店钩子设计与转化路径（脱敏示范）

> 核心：**钩子是请客户进门的理由，不是免费午餐。** 引流款必须自带"防白嫖门槛 + 接面诊路径"，否则只引来薅羊毛的人。
> （价格/项目名为占位，客户填自己的真实设置）

## 钩子三类（按目的选）
| 钩子类型 | 示例 | 目的 | 防白嫖设计 |
|---|---|---|---|
| 引流款（低价体验）| 〔皮肤检测〕/〔小气泡体验价〕 | 拉到店 | 限新客、到店才享、需预约面诊 |
| 爆款团购券 | 〔爆款项目名〕团购套餐 | 放量转化 | 到店核销、绑定升单方案 |
| 到店礼 | 到店即送〔到店礼〕 | 提高到店率 | 仅核销当天领、不可转赠 |

## 防"只引白嫖党"四招
1. **设门槛**：引流款限新客、限量、需预约，不做人人可领的纯薅羊毛入口。
2. **绑面诊**：低价项目必须配一次到店面诊（皮肤检测/方案咨询），让客户和医师真实接触。
3. **价值锚点**：体验款讲清"原价 vs 体验价"，凸显这是体验机会不是廉价处理。
4. **升单路径**：每个引流款都预设"体验 → 面诊 → 正价项目"的衔接动线，引来就有下一步。

## 转化路径（公域钩子 → 到店 → 面诊）
```
公域钩子(引流款/团购/到店礼)
        ↓  预约/核销（埋追踪标识）
到店核销 → 一次到店面诊（皮肤检测/方案咨询）
        ↓  留资即转
交接 [[在线咨询到到店转化SOP]]（面诊转化工作流）
```

## 合规与隐私（必须遵守）
- 钩子文案禁疗效保证、禁绝对化用语、禁用前后对比图诱导；涉及项目须可追溯医师资质。
- 客户到店产生的术前照片/检测记录/病历 = 🔴 医疗隐私受限，绝不外发、输出必打码。

## 用法
先按目的选钩子类型，再套防白嫖四招，最后确认升单路径接得上面诊。到店客户 → 交 [[在线咨询到到店转化SOP]]。

# 医美三平台种草矩阵（脱敏示范）

> 核心：**三平台不是重复发同一条，是漏斗的三段分工。** 小红书拦截搜索、抖音放量、大众点评临门一脚。
> （价格/项目名为占位，客户填自己的真实设置）

| 平台 | 漏斗定位 | 主打内容形态 | 引流终点 |
|---|---|---|---|
| 小红书 | 口碑种草 + 搜索拦截 | 真人变美日记/项目科普/避坑攻略/探店笔记 | 私域（评论/主页钩子）|
| 抖音 | 泛流量 + 本地生活团购 | 短视频科普/医师出镜/团购挂车 | 团购核销 → 到店 |
| 大众点评 | 到店决策临门一脚 | 机构主页/真实评价/星级/团购套餐 | 直接到店核销 |

## 各平台打法清单
- **小红书**：铺"项目名+城市/机构"长尾关键词，拦截主动搜索的高意向人群；笔记结尾引私域（"主页领〔皮肤检测〕名额"），靠口碑量堆信任。
- **抖音**：医师/真人出镜做科普涨泛流量，挂〔团购体验价〕做本地生活转化；直播间承接，团购券导向到店。
- **大众点评**：维护机构主页评分与真实评价（决策最后一公里看这里），上架〔团购套餐〕，让搜索到的人直接下单到店。

## 三平台共同合规底线（必须遵守）
1. **禁前后对比图诱导**：不得用术前术后对比图作引流卖点（违规 + 涉医疗隐私）。
2. **禁疗效/绝对化话术**：不写"一次见效""包好""永久""最佳"等，不承诺疗效。
3. **必标资质**：涉及医疗项目须标注医疗机构资质，项目可追溯医师资质，不确定→人工确认。

## 用法
先定这次主攻哪一层平台，再选对应内容形态。三平台引来的留资/到店客户 → 统一交 [[到店钩子设计与转化路径]] 衔接面诊。

# 权限分级 · 四级

> 企业库和个人库最硬的区别。没有权限治理 = AI 资料垃圾场 + 隐私事故。

| 级别 | 含义 | 典型资产 | agent 行为 |
|---|---|---|---|
| 🟢 公开 | 可对外发布 | 对外物料、方法论、介绍 | 可写入对外产物 |
| 🟡 内部 | 团队内用 | 话术、SOP、运营数据 | 内部可用,不对外 |
| 🔴 受限 | 隐私/敏感 | 客户对话、通话记录、手机号、名单 | **绝不写入对外产物,只内部分析** |
| ⛔ 禁止入库 | 不该进库 | 证件、银行信息、未授权第三方资料 | 拒绝入库 |

## agent 红线
1. 🔴 受限资产 → 输出时必须脱敏/打码,绝不原文外发
2. 超标准的报价/优惠/对外承诺 → 必须人工确认
3. 不确定权限 → 默认按更高一级处理,问用户

# 医美行业 · 合规与风控默认值（行业特有，权重最高）

> 合成进成品库 `90-governance/行业合规默认.md`。医美默认更严，客户只可收紧不可放松。

## 默认权限分级（医美收紧版）
| 资产类型 | 默认级别 |
|---|---|
| 客户病历/术前术后照片/诊疗记录 | 🔴 受限（医疗隐私，永不对外、输出必脱敏打码）|
| 客户个人信息（姓名/手机/微信）| 🔴 受限 |
| 医师资质/机构许可/药械证照 | ⛔ 禁止入库（证件类，留原地索引）|
| 项目话术/咨询SOP/运营数据 | 🟡 内部 |
| 对外科普/品牌介绍 | 🟢 公开（须经合规审核，禁违反医疗广告法）|

## 医美红线（agent 必须遵守）
1. **医疗广告合规**：对外内容不得宣称疗效保证、不得用前后对比图诱导、不得绝对化用语。
2. **诊疗隐私**：术前术后照片/病历属医疗隐私，🔴 受限，绝不进定版、绝不对外、输出必打码。
3. **资质追溯**：涉及项目须可追溯医师资质与药械合规，不确定→人工确认。
4. **纠纷处理**：客诉/纠纷走 `compliance-risk` 工作流，高风险动作（赔付/对外声明）必须人工把关。

> 合规与风控是医美独立工作流 `compliance-risk`，不是附属——这是医美区别于其他行业的核心。
