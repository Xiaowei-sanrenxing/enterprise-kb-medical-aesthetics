# 合规与风控 SOP · 人可读总览 ★医美核心

> 给人看的全流程。Agent 执行请走 [`_task-card.md`](_task-card.md)。
> 医美第一权重：合规前置，高风险动作人工把关，纠纷分级处理。

## 一句话流程
先分流（上新发布 / 客诉纠纷）→ 上新先核资质再过广告合规、经审查留痕才发 → 客诉先定级、agent 只处理最轻级、其余必转人工 → 隐私全程🔴受限脱敏 → 高风险动作一律人工+法务/医师。

## 铁律（本工作流最严，逐条不可破）
1. **资质可追溯，超范围零容忍**：机构《医疗机构执业许可证》、医师资格证+执业证+执业范围、药械正规来源齐全且对得上，缺一或超范围 → 🛑 停，不得发布/上新。
2. **医疗广告不越线**：禁疗效保证、禁治愈率/有效率、禁绝对化用语（最/第一/100%）、禁前后对比图诱导、禁用患者形象作证明、须经审查、严格区分"医疗"与"生活美容"。
3. **隐私永不外发**：术前术后照片/病历/诊疗记录/客户个人信息 = 医疗隐私🔴受限，绝不进定版、绝不对外、对外输出必脱敏打码，授权使用须书面同意。
4. **纠纷分级，agent 只碰最轻一级**：仅"轻度不满意"可由 agent 安抚+登记+上报；效果争议/疑似不良反应/舆情/诉讼 → 立即人工。
5. **高风险动作 agent 绝不决策**：赔付、对外声明、舆情回应、超范围放行、医疗广告放行 → 强制人工+法务/医师把关。
6. **不确定就停**：任何拿不准 → 默认拦截并上报〔合规负责人〕，宁可拦错不可放过。

# 合同/协议/知情同意书 条款合规清单

> 补盲区：疗效红线此前全锚定"对外广告"，本清单专管**合同/协议/知情同意书的文本措辞**——医美纠纷最高发区。
> 核心：写进合同的效果承诺，比口头更硬地踩疗效保证红线，更要管住。

## 🔴 禁写条款（一律不得写入合同/协议/同意书）
- "保证/一定/彻底解决/根治/100% 有效/永久" 等效果保证或绝对化承诺
- 任何量化疗效承诺（"有效率 X%""X 次见效"）
- 与事实不符的资质/技术描述

## ✅ 可写 / 应写条款（合规且保护双方）
- **效果表述**：用"预期改善/因人而异/最终效果以个体为准"，不写保证。
- **知情同意**：项目原理、风险、并发症、术后注意、替代方案——须充分告知并签字（书面留痕）。
- **退费规则**：明确未开始服务的退费比例、已部分履约的结算方式、冷静期（按当地消保/合同法），**用"服务履约"口径而非"效果不达标包退"**。
- **争议解决**：约定协商/投诉/调解路径。

## 关于"效果不达标退费"
医美效果受个体差异影响，**不建议承诺"无效退款/效果不达标全退"**（等于变相疗效保证，易引纠纷）。如客户坚持，转法务定制条款，agent 不自拟。

## agent 动作
1. 客户/老板要求把效果保证写进合同 → 拒绝写入，解释属书面疗效承诺红线，导向"预期改善+知情同意+履约退费"合规写法。
2. 任何合同/协议/同意书定稿 → **必须法务审核 + 人工确认**，agent 不自行出具有法律效力的文本。
3. 不确定某条款 → 转法务，默认从严。

> 核心：**隐私永不外发、纠纷分级处理、高风险动作 agent 零决策。**
> agent 只能处理最轻一级纠纷；赔付、对外声明、舆情回应绝不可由 agent 决定。

## 一、隐私保护铁律（处理任何客户数据前必读）

医疗隐私范围 = 术前术后照片 + 病历 + 诊疗记录 + 客户个人信息（姓名/手机/微信/身份信息）= 🔴 受限。

- **存储分区**：隐私数据独立分区存放，与运营/对外内容物理或逻辑隔离，不混入定版资产。
- **最小授权**：按岗位最小权限访问，agent 默认无权读取原始隐私数据，只读脱敏后结果。
- **对外必脱敏**：任何对外输出（科普/案例/朋友圈/平台）一律脱敏打码——面部打码、去标识、不可反推到个人。
- **授权须书面**：使用客户照片/案例对外，必须取得**书面同意**（明确用途、范围、期限），无书面授权 = 绝不使用。
- **永不进定版**：隐私数据绝不进定版库、绝不对外、绝不写入任何对外产物。

## 二、纠纷分级处理表（agent 先定级，再按级响应）

| 级别 | 典型情形 | 响应时限 | 谁处理 | agent 可否处理 | 可否对外回应 |
|---|---|---|---|---|---|
| 🟢 L1 轻度不满意 | 等待久、态度、轻微体验抱怨、效果"没那么明显"的情绪表达 | 即时 | 客服/运营 | ✅ **可**：安抚+登记+上报 | 仅安抚，不承诺 |
| 🟡 L2 效果争议 | 客户主张效果不达预期、要求退费/重做 | 当日 | 〔客服主管〕+ 医师评估 | ❌ 登记后立即转人工 | ❌ |
| 🟠 L3 疑似不良反应/医疗事故 | 红肿溃烂、感染、损伤、疑似事故 | 立即 | **医师 + 法务 + 合规负责人** | ❌ 立即移交，不回应不处置 | ❌ 严禁 |
| 🔴 L4 媒体舆情 | 平台曝光、媒体询问、负面扩散 | 立即 | **法务 + 公关 + 合规负责人** | ❌ 立即上报 | ❌ 仅法务/公关口径 |
| ⚫ L5 法律诉讼 | 投诉至监管、律师函、起诉 | 立即 | **法务主导 + 合规负责人** | ❌ 立即上报 | ❌ 仅法务口径 |

### agent 处理边界（硬约束）
- **只能处理 🟢 L1**：动作 = 安抚情绪 + 据实登记到〔投诉登记表〕 + 上报，不承诺退赔、不下结论。
- **L2~L5 一律转人工**：agent 登记+升级通知后即停手，不再自行回应。
- **🛑 绝不决策**：赔付金额、对外/媒体声明、舆情回应、是否构成事故 —— 全部由人工+法务/医师决定，agent 零决策权。
- **拿不准级别 → 就高不就低**：宁可升级，不可压级。

## 用法
处理客诉先定级 → L1 自己闭环（安抚+登记+上报）→ L2 及以上立即升级对应人工并停手。涉及照片/病历等隐私一律按上方铁律脱敏处理。对外内容合规与资质核验见 [[资质核验与医疗广告合规清单]]。

# 竞品诋毁 / 不正当竞争 处置路径

> 补盲区：原纠纷分级只覆盖"自家被客户/媒体负面"的被动舆情，本流程专管**竞品主动诋毁/虚假比较/不正当竞争**（属《反不正当竞争法》范畴，不是医疗广告/客诉），并给出**主动反制的合规入口**。

## 第一判断：是哪一类
| 类型 | 信号 | 走向 |
|---|---|---|
| 客户负面/媒体舆情 | 真实客户投诉、媒体报道 | 走 [[客户隐私保护与纠纷分级处理]] 舆情分级 |
| **竞品诋毁/不正当竞争** | 竞品/水军捏造、虚假比较、贬低我方 | 走本流程 |

## 处置三步（合规反制，agent 不自行对线）
1. **取证固定**：第一时间截图/录屏/公证留存诋毁内容（时间、账号、链接），不删不回怼。
2. **走合规渠道反制**（由法务/公关主导，agent 只协助整理材料）：
   - 平台投诉/举报（抖音等平台的不正当竞争/诋毁举报通道）
   - 法务评估发律师函 / 必要时诉讼
   - 公关统一口径回应（如需）
3. **上报**：立即上报法务 + 公关 + 合规负责人，按高风险动作零决策处理。

## 🔴 agent 绝不做
- 不自行在评论区/公开平台和竞品对线、互撕
- 不自行发声明、不自行挂"律师函截图"（对外声明/法律动作属零决策权高风险动作，必须法务）
- 不捏造或夸大反击对方（我方反制也须合法，不得变成新的不正当竞争）

## agent 标准回应（老板问"要不要发声明/挂律师函"时）
"这属于竞品不正当竞争，建议：① 先取证固定（截图/公证）② 走平台投诉 + 法务评估发函，③ 对外声明须法务/公关定稿。我不能自行发声明或挂律师函截图，已帮你把材料整理好转法务。"

# 资质核验与医疗广告合规清单（医美核心）★

> 核心：**发布对外内容 / 上新项目前，先过这两道闸门。任一项勾不掉 → 🛑 停，转〔合规负责人〕，不得放行。**
> 法规以《医疗广告管理办法》《广告法》及主管部门要求为准（此处讲要点，具体条款以官方文本为准）。

## 一、资质核验清单（上新 / 上项目前，agent 逐项核对，缺一即拦）

| 核验项 | 要点 | 不达标动作 |
|---|---|---|
| ☐ 机构资质 | 〔机构名〕持有效《医疗机构执业许可证》，诊疗科目覆盖该项目 | 🛑 停，转〔合规负责人〕 |
| ☐ 医师资质 | 操作医师持《医师资格证》+《医师执业证》，且**执业范围**覆盖该项目 | 🛑 停，超范围零容忍 |
| ☐ 药械来源 | 注射类/光电类药械经正规渠道采购、有合法批准文号、可溯源 | 🛑 停，禁用来源不明药械 |
| ☐ 无超范围操作 | 项目不超出机构诊疗科目与医师执业范围 | 🛑 超范围零容忍，绝不放行 |

> 证照原件 = ⛔禁止入库（证件类，留原地索引）；agent 只读核验结论，不留存证件影像。

## 二、医疗广告合规自检清单（对外内容发布前，逐条勾选，任一勾不掉 → 改稿或拦截）

- ☐ 无**疗效保证**（不写"保证有效/必定见效/无效退款"等承诺）
- ☐ 无**治愈率 / 有效率 / 成功率**等数字承诺
- ☐ 无**绝对化用语**（最/第一/顶级/100%/唯一/根治）
- ☐ 无**前后对比图**诱导（术前 vs 术后对比图禁用）
- ☐ 不**利用患者形象 / 名义 / 评价作证明**（禁患者现身说法、禁好评截图当广告）
- ☐ 已**送医疗广告审查**并取得审查通过（医疗广告须先审后发）
- ☐ 严格区分"**医疗美容**"与"**生活美容**"，不把医疗项目包装成生活美容规避监管
- ☐ 无夸大、无误导、无与同行作贬低性比较

### 红线表（命中即拦截，无例外）
| 红线 | 典型违规话术（示例，禁用） |
|---|---|
| 疗效保证 | "包好""无效退款""一次见效永久不反弹" |
| 数字承诺 | "有效率 99%""治愈率行业第一" |
| 绝对化 | "全市最好""第一品牌""100% 安全无风险" |
| 前后对比 | 配"术前→术后"对比图诱导下单 |
| 患者证言 | "顾客实拍好评""真人现身说法" |

## 用法
agent 发布对外内容 / 上新项目前，先跑「资质核验」再跑「广告合规自检」；两道全过 + 经〔合规负责人〕审查留痕 → 才可放行。任何不确定 → 默认拦截上报。隐私数据处理与纠纷应对见 [[客户隐私保护与纠纷分级处理]]。

# 权限分级 · 四级

> 企业库和个人库最硬的区别。没有权限治理 = AI 资料垃圾场 + 隐私事故。

| 级别 | 含义 | 典型资产 | agent 行为 |
|---|---|---|---|
| 🟢 公开 | 可对外发布 | 对外物料、方法论、介绍 | 可写入对外产物 |
| 🟡 内部 | 团队内用 | 话术、SOP、运营数据 | 内部可用,不对外 |
| 🔴 受限 | 隐私/敏感 | 客户对话、通话记录、手机号、名单 | **绝不写入对外产物,只内部分析** |
| ⛔ 禁止入库 | 不该进库 | 证件、银行信息、未授权第三方资料 | 拒绝入库 |

## agent 红线
1. 🔴 受限资产 → 输出时必须脱敏/打码,绝不原文外发
2. 超标准的报价/优惠/对外承诺 → 必须人工确认
3. 不确定权限 → 默认按更高一级处理,问用户


## 治理 · 行业合规默认.md

# 医美行业 · 合规与风控默认值（行业特有，权重最高）

> 合成进成品库 `90-governance/行业合规默认.md`。医美默认更严，客户只可收紧不可放松。

## 默认权限分级（医美收紧版）
| 资产类型 | 默认级别 |
|---|---|
| 客户病历/术前术后照片/诊疗记录 | 🔴 受限（医疗隐私，永不对外、输出必脱敏打码）|
| 客户个人信息（姓名/手机/微信）| 🔴 受限 |
| 医师资质/机构许可/药械证照 | ⛔ 禁止入库（证件类，留原地索引）|
| 项目话术/咨询SOP/运营数据 | 🟡 内部 |
| 对外科普/品牌介绍 | 🟢 公开（须经合规审核，禁违反医疗广告法）|

## 医美红线（agent 必须遵守）
1. **医疗广告合规**：对外内容不得宣称疗效保证、不得用前后对比图诱导、不得绝对化用语。
2. **诊疗隐私**：术前术后照片/病历属医疗隐私，🔴 受限，绝不进定版、绝不对外、输出必打码。
3. **资质追溯**：涉及项目须可追溯医师资质与药械合规，不确定→人工确认。
4. **纠纷处理**：客诉/纠纷走 `compliance-risk` 工作流，高风险动作（赔付/对外声明）必须人工把关。

> 合规与风控是医美独立工作流 `compliance-risk`，不是附属——这是医美区别于其他行业的核心。
