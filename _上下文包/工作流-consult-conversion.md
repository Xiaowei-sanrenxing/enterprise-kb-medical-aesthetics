# 任务卡 · 咨询面诊转化（consult-conversion）· 脱敏示范

> Agent 操作总台。接到"在线咨询/挖需求/邀约到店/面诊出方案/异议处理"类任务,从这里开始。
> 医美红线最高优先:线上不诊断、不报价、不承诺效果。

## 🎯 目标(Goal)
把一个在线咨询线索,从首次触达推进到到店面诊并成交,或记录明确流失原因,全程守住"线上不诊断/不报价/不承诺疗效"的合规边界。

## 📥 上下文(Context) —— 动手前必读全
1. [`context/在线咨询到到店转化SOP.md`](context/在线咨询到到店转化SOP.md) — 全链路阶段动作 + 合规边界
2. [`context/客户分级与异议处理.md`](context/客户分级与异议处理.md) — 判级标准 + 高频异议应对
3. [`context/未成年咨询处置流程.md`](context/未成年咨询处置流程.md) — 疑似未成年：核龄→须监护人到场→书面同意，缺即停 + 家长缺位话术
4. 〔咨询话术库索引〕 — 按场景查标准话术(真实库含此项)
5. 〔项目方案与定价规则〕 — 仅面诊后由咨询师/医师出(真实库含此项)

## 🪜 指令(Instructions)
1. 黄金窗口内首回(示意 ≤ 3 分钟),开放式开场(禁"在吗"/二选一)
2. 挖需求:关注部位 / 预算区间 / 核心顾虑 → 初判等级
3. 线上只科普共性知识,**不诊断、不报价、不承诺效果**
4. 用邀约钩子把对话导向到店面诊(免费面诊/医师亲诊/限时名额示意)
5. 到店:咨询师承接 → 医师面诊判断 → 出个性化方案 → 报价成交
6. 异议按结构应对(给真实预期、强调资质与安全,不承诺效果)
7. 成交或记录流失原因 → 回流自我进化接口

## 🛠 工具(Tools)
〔CRM/客户档案读写〕 · 咨询话术库检索 · 〔到店预约系统〕 · 〔面诊方案模板〕

## 🔒 权限(Permission)
- 可读:咨询SOP、分级、异议库、对外科普话术
- 🔴 受限:客户聊天原文/手机号/微信号/术前照 → 医疗隐私,永不外发、输出必脱敏打码
- 线上禁止:诊断、报价、疗效保证、绝对化用语(如"一定有效""零风险")
- 须人工确认:超标准报价优惠、涉及项目的医师资质追溯、未成年人医美(须监护人同意方可推进)

## ✅ 验收(Acceptance)
- 首回 ≤ 黄金窗口 ｜ 开场开放式 ｜ 线上无诊断/无报价/无疗效承诺 ｜ 邀约到店有钩子
- 隐私零外泄 ｜ 未成年线索已核监护人同意 ｜ 涉项目资质可追溯 ｜ 流失有记录原因
- 详见 evals.md

## 🔄 自我进化接口
每条咨询记录"成交/流失 + 卡在哪阶段 + 触发了哪类异议" → 回流诊断数据 → 监控 agent 优化话术与邀约钩子,违规话术(诊断/报价/承诺)进黑名单。

# 咨询面诊转化 SOP · 人可读总览

> 给人看的全流程。Agent 执行请走 [`_task-card.md`](_task-card.md)。
> 医美核心:线上是"挖需求 + 建信任 + 邀到店",诊断与定价一律到店面诊完成。

## 一句话流程
黄金窗口温度开场 → 挖需求(部位/预算/顾虑)初判级 → 线上只科普不诊断不报价 → 邀约到店面诊 → 咨询师承接+医师面诊出方案 → 报价处理异议 → 成交/记录流失。

## 几条铁律
1. **线上三不**:不诊断、不报价、不承诺效果——这是医美合规红线,也是引导到店的钩子。
2. **黄金窗口首回**:首回越慢,流失越多;开场禁"在吗"、禁二选一,用开放式问句挖需求。
3. **报价只在面诊后**:医美按个性化方案定价,线上轻易报价=失客+合规风险。
4. **隐私红线**:客户对话/手机/微信/术前照 🔴 受限,绝不外发、输出必打码。
5. **资质可追溯**:涉及项目须可追溯医师资质与药械合规,不确定→人工确认。
6. **未成年须监护人**:未成年人医美须监护人到场同意,不得绕过。

## 详细资产
- [`context/在线咨询到到店转化SOP.md`](context/在线咨询到到店转化SOP.md) — 全链路阶段动作 + 合规边界
- [`context/客户分级与异议处理.md`](context/客户分级与异议处理.md) — 判级标准 + 高频异议应对
- 〔咨询话术库索引〕 — 客户补:按场景查标准话术

# 在线咨询到到店转化 SOP（脱敏示范）

> 核心:**线上不诊断、不报价、不承诺效果——线上的任务是挖需求、建信任、邀到店;诊断与定价一律到店面诊完成。**
> (响应时长/名额数为示意,客户填自己的真实设置)

## 阶段化清单

### 阶段一 · 黄金响应(线上)
- [ ] 黄金窗口内首回(示意 ≤ 3 分钟),开放式开场,禁"在吗"、禁二选一
- [ ] 30 秒看来源/资料初判等级(详见 [[客户分级与异议处理]])

### 阶段二 · 首咨挖需求(线上)
- [ ] 开放式提问挖三件事:**关注部位 / 预算区间 / 核心顾虑**
- [ ] 只科普共性知识,**不对个体下诊断**(如"你这个适合做XX")
- [ ] 客户问价 → 不报价,说明医美按个性化方案定价,价格需面诊后定

### 阶段三 · 合规边界(贯穿线上全程)⚠️
- [ ] **不诊断**:不替医师判断适应症/方案
- [ ] **不报价**:不在线上给具体项目价格/优惠承诺
- [ ] **不承诺效果**:禁"一定有效/保证变美/零风险/永久"等疗效与绝对化用语
- [ ] 客户聊天/手机/微信/术前照 = 🔴 医疗隐私,永不外发、输出必打码

### 阶段四 · 邀约到店(线上→线下)
- [ ] 给钩子:免费面诊 / 医师亲诊 / 限时面诊名额(示意)
- [ ] 话术结构:**共情顾虑 → 给到店独有价值 → 给具体时间二选一约**
- [ ] 约定到店时间 → 进 〔到店预约系统〕,提前提醒

### 阶段五 · 到店面诊(线下)
- [ ] **咨询师**承接接待、了解诉求、建立信任
- [ ] **医师**面诊判断适应症、给个性化方案(此处才出诊断)
- [ ] 涉及项目须可追溯医师资质与药械合规,不确定→人工确认
- [ ] 未成年人医美须监护人到场同意

### 阶段六 · 出方案与成交(线下)
- [ ] 咨询师据医师方案出报价,逐一处理异议(见 [[客户分级与异议处理]])
- [ ] 成交 / 记录流失原因 → 回流自我进化接口

# 客户分级与异议处理（脱敏示范）

> 核心:**先判级再打法;所有异议应对都守住"不承诺效果、给真实预期、强调资质与安全"。**
> (打法为示意,客户按真实话术库补充)

## 一、客户分级与打法

| 等级 | 判级标准 | 对应打法 |
|---|---|---|
| 高意向 | 部位明确、预算清晰、主动问到店/时间 | 快速邀约到店面诊,医师亲诊出方案,趁热成交 |
| 比价型 | 反复问价、只比价格、不谈需求 | **不卷价格**,转价值/资质/安全锚点,邀到店体验差异 |
| 犹豫型 | 有需求但反复纠结、回复慢 | 给真实预期降焦虑,限时面诊钩子推一把,转长期触达 |
| 纠结安全型 | 高度关注疼痛/副作用/风险/真假 | 重点讲资质+安全+真实案例(脱敏),邀面诊当面释疑 |
| 咨询白嫖型 | 只要免费攻略、无到店意愿 | 给共性科普即可,不投入深聊,标记低优先 |

## 二、高频异议应对结构

| 异议 | 标准应对结构(不承诺效果) |
|---|---|
| 怕疼 | 共情 → 说明麻醉/舒适化手段 → 给真实痛感预期 → 邀面诊体验 |
| 怕假/不自然 | 强调医师审美与资质 → 给真实自然预期(非"绝对自然")→ 看脱敏案例 |
| 怕副作用 | **不说"零风险"** → 说明正规药械与风险告知 → 强调资质保障 → 邀面诊评估 |
| 比价 | 不贬低同行 → 锚定资质/医师/药械正品/术后保障 → 邀到店感受差异 |
| 老公/家人不同意 | 共情 → 提供可对家人解释的客观信息 → 不施压 → 留长期触达 |

## 合规提醒 ⚠️
- 任何应对都**禁疗效承诺、禁夸大效果、禁绝对化**(如"一定有效""零风险""永久")
- 引用案例须脱敏,术前照属 🔴 医疗隐私不得外发
- 涉及适应症判断 → 一律导向医师面诊,不在线上诊断

# 未成年咨询处置流程（操作 SOP · 三步缺即停）

> 补"须监护人同意"红线的盲区：原则在，缺"家长缺位/不知情怎么办、是否书面、年龄怎么核"的操作动作。
> 核心：未成年医美**不是不能做，而是必须监护人到场书面同意**；任何"绕过家长尽快约"的诉求一律拒绝。

## 三步流程（缺任一即停，不得推进成交）
1. **核龄**：咨询中识别疑似未成年（学生/年龄信号）→ 主动、礼貌确认真实年龄；不得为成交装看不见。未满 18 岁 = 进入本流程。
2. **须监护人到场**：明确告知"未成年医美项目须监护人陪同到场、共同知情同意后方可评估"。线上只做科普答疑，**不报价、不诊断、不预约手术档**，可登记意向但状态标"待监护人确认"。
3. **书面同意留痕**：监护人到场后，须签署知情同意（书面留痕，参照隐私/知情铁律），核验监护关系。无书面 = 不推进。

## 家长缺位/不知情场景标准话术（agent 用）
- 客户说"家长不会同意/不想让家长知道" → 明确而温和拒绝："这类项目按规定必须监护人陪同同意，我们不能绕过哦。你可以先了解信息，等和家长沟通好、一起过来，我们再给专业评估。"
- 客户催"能不能先排上/尽快约" → "登记可以，但要监护人到场确认后才能正式安排，这是对你安全负责。"
- **绝不**：为"尽快/坚决"绕过监护人、不暗示"瞒着家长也能做"。

## 红线
- 🔴 未成年无监护人到场同意 → 不评估、不报价、不成交
- 🔴 不诱导未成年绕过监护人

# 权限分级 · 四级

> 企业库和个人库最硬的区别。没有权限治理 = AI 资料垃圾场 + 隐私事故。

| 级别 | 含义 | 典型资产 | agent 行为 |
|---|---|---|---|
| 🟢 公开 | 可对外发布 | 对外物料、方法论、介绍 | 可写入对外产物 |
| 🟡 内部 | 团队内用 | 话术、SOP、运营数据 | 内部可用,不对外 |
| 🔴 受限 | 隐私/敏感 | 客户对话、通话记录、手机号、名单 | **绝不写入对外产物,只内部分析** |
| ⛔ 禁止入库 | 不该进库 | 证件、银行信息、未授权第三方资料 | 拒绝入库 |

## agent 红线
1. 🔴 受限资产 → 输出时必须脱敏/打码,绝不原文外发
2. 超标准的报价/优惠/对外承诺 → 必须人工确认
3. 不确定权限 → 默认按更高一级处理,问用户


## 治理 · 行业合规默认.md

# 医美行业 · 合规与风控默认值（行业特有，权重最高）

> 合成进成品库 `90-governance/行业合规默认.md`。医美默认更严，客户只可收紧不可放松。

## 默认权限分级（医美收紧版）
| 资产类型 | 默认级别 |
|---|---|
| 客户病历/术前术后照片/诊疗记录 | 🔴 受限（医疗隐私，永不对外、输出必脱敏打码）|
| 客户个人信息（姓名/手机/微信）| 🔴 受限 |
| 医师资质/机构许可/药械证照 | ⛔ 禁止入库（证件类，留原地索引）|
| 项目话术/咨询SOP/运营数据 | 🟡 内部 |
| 对外科普/品牌介绍 | 🟢 公开（须经合规审核，禁违反医疗广告法）|

## 医美红线（agent 必须遵守）
1. **医疗广告合规**：对外内容不得宣称疗效保证、不得用前后对比图诱导、不得绝对化用语。
2. **诊疗隐私**：术前术后照片/病历属医疗隐私，🔴 受限，绝不进定版、绝不对外、输出必打码。
3. **资质追溯**：涉及项目须可追溯医师资质与药械合规，不确定→人工确认。
4. **纠纷处理**：客诉/纠纷走 `compliance-risk` 工作流，高风险动作（赔付/对外声明）必须人工把关。

> 合规与风控是医美独立工作流 `compliance-risk`，不是附属——这是医美区别于其他行业的核心。
