# 合规与风控 SOP · 人可读总览 ★医美核心

> 给人看的全流程。Agent 执行请走 [`_task-card.md`](_task-card.md)。
> 医美第一权重：合规前置，高风险动作人工把关，纠纷分级处理。

## 一句话流程
先分流（上新发布 / 客诉纠纷）→ 上新先核资质再过广告合规、经审查留痕才发 → 客诉先定级、agent 只处理最轻级、其余必转人工 → 隐私全程🔴受限脱敏 → 高风险动作一律人工+法务/医师。

## 铁律（本工作流最严，逐条不可破）
1. **资质可追溯，超范围零容忍**：机构《医疗机构执业许可证》、医师资格证+执业证+执业范围、药械正规来源齐全且对得上，缺一或超范围 → 🛑 停，不得发布/上新。
2. **医疗广告不越线**：禁疗效保证、禁治愈率/有效率、禁绝对化用语（最/第一/100%）、禁前后对比图诱导、禁用患者形象作证明、须经审查、严格区分"医疗"与"生活美容"。
3. **隐私永不外发**：术前术后照片/病历/诊疗记录/客户个人信息 = 医疗隐私🔴受限，绝不进定版、绝不对外、对外输出必脱敏打码，授权使用须书面同意。
4. **纠纷分级，agent 只碰最轻一级**：仅"轻度不满意"可由 agent 安抚+登记+上报；效果争议/疑似不良反应/舆情/诉讼 → 立即人工。
5. **高风险动作 agent 绝不决策**：赔付、对外声明、舆情回应、超范围放行、医疗广告放行 → 强制人工+法务/医师把关。
6. **不确定就停**：任何拿不准 → 默认拦截并上报〔合规负责人〕，宁可拦错不可放过。

# 合同/协议/知情同意书 条款合规清单

> 补盲区：疗效红线此前全锚定"对外广告"，本清单专管**合同/协议/知情同意书的文本措辞**——医美纠纷最高发区。
> 核心：写进合同的效果承诺，比口头更硬地踩疗效保证红线，更要管住。

## 🔴 禁写条款（一律不得写入合同/协议/同意书）
- "保证/一定/彻底解决/根治/100% 有效/永久" 等效果保证或绝对化承诺
- 任何量化疗效承诺（"有效率 X%""X 次见效"）
- 与事实不符的资质/技术描述

## ✅ 可写 / 应写条款（合规且保护双方）
- **效果表述**：用"预期改善/因人而异/最终效果以个体为准"，不写保证。
- **知情同意**：项目原理、风险、并发症、术后注意、替代方案——须充分告知并签字（书面留痕）。
- **退费规则**：明确未开始服务的退费比例、已部分履约的结算方式、冷静期（按当地消保/合同法），**用"服务履约"口径而非"效果不达标包退"**。
- **争议解决**：约定协商/投诉/调解路径。

## 关于"效果不达标退费"
医美效果受个体差异影响，**不建议承诺"无效退款/效果不达标全退"**（等于变相疗效保证，易引纠纷）。如客户坚持，转法务定制条款，agent 不自拟。

## agent 动作
1. 客户/老板要求把效果保证写进合同 → 拒绝写入，解释属书面疗效承诺红线，导向"预期改善+知情同意+履约退费"合规写法。
2. 任何合同/协议/同意书定稿 → **必须法务审核 + 人工确认**，agent 不自行出具有法律效力的文本。
3. 不确定某条款 → 转法务，默认从严。

> 核心：**隐私永不外发、纠纷分级处理、高风险动作 agent 零决策。**
> agent 只能处理最轻一级纠纷；赔付、对外声明、舆情回应绝不可由 agent 决定。

## 一、隐私保护铁律（处理任何客户数据前必读）

医疗隐私范围 = 术前术后照片 + 病历 + 诊疗记录 + 客户个人信息（姓名/手机/微信/身份信息）= 🔴 受限。

- **存储分区**：隐私数据独立分区存放，与运营/对外内容物理或逻辑隔离，不混入定版资产。
- **最小授权**：按岗位最小权限访问，agent 默认无权读取原始隐私数据，只读脱敏后结果。
- **对外必脱敏**：任何对外输出（科普/案例/朋友圈/平台）一律脱敏打码——面部打码、去标识、不可反推到个人。
- **授权须书面**：使用客户照片/案例对外，必须取得**书面同意**（明确用途、范围、期限），无书面授权 = 绝不使用。
- **永不进定版**：隐私数据绝不进定版库、绝不对外、绝不写入任何对外产物。

## 二、纠纷分级处理表（agent 先定级，再按级响应）

| 级别 | 典型情形 | 响应时限 | 谁处理 | agent 可否处理 | 可否对外回应 |
|---|---|---|---|---|---|
| 🟢 L1 轻度不满意 | 等待久、态度、轻微体验抱怨、效果"没那么明显"的情绪表达 | 即时 | 客服/运营 | ✅ **可**：安抚+登记+上报 | 仅安抚，不承诺 |
| 🟡 L2 效果争议 | 客户主张效果不达预期、要求退费/重做 | 当日 | 〔客服主管〕+ 医师评估 | ❌ 登记后立即转人工 | ❌ |
| 🟠 L3 疑似不良反应/医疗事故 | 红肿溃烂、感染、损伤、疑似事故 | 立即 | **医师 + 法务 + 合规负责人** | ❌ 立即移交，不回应不处置 | ❌ 严禁 |
| 🔴 L4 媒体舆情 | 平台曝光、媒体询问、负面扩散 | 立即 | **法务 + 公关 + 合规负责人** | ❌ 立即上报 | ❌ 仅法务/公关口径 |
| ⚫ L5 法律诉讼 | 投诉至监管、律师函、起诉 | 立即 | **法务主导 + 合规负责人** | ❌ 立即上报 | ❌ 仅法务口径 |

### agent 处理边界（硬约束）
- **只能处理 🟢 L1**：动作 = 安抚情绪 + 据实登记到〔投诉登记表〕 + 上报，不承诺退赔、不下结论。
- **L2~L5 一律转人工**：agent 登记+升级通知后即停手，不再自行回应。
- **🛑 绝不决策**：赔付金额、对外/媒体声明、舆情回应、是否构成事故 —— 全部由人工+法务/医师决定，agent 零决策权。
- **拿不准级别 → 就高不就低**：宁可升级，不可压级。

## 用法
处理客诉先定级 → L1 自己闭环（安抚+登记+上报）→ L2 及以上立即升级对应人工并停手。涉及照片/病历等隐私一律按上方铁律脱敏处理。对外内容合规与资质核验见 [[资质核验与医疗广告合规清单]]。

# 竞品诋毁 / 不正当竞争 处置路径

> 补盲区：原纠纷分级只覆盖"自家被客户/媒体负面"的被动舆情，本流程专管**竞品主动诋毁/虚假比较/不正当竞争**（属《反不正当竞争法》范畴，不是医疗广告/客诉），并给出**主动反制的合规入口**。

## 第一判断：是哪一类
| 类型 | 信号 | 走向 |
|---|---|---|
| 客户负面/媒体舆情 | 真实客户投诉、媒体报道 | 走 [[客户隐私保护与纠纷分级处理]] 舆情分级 |
| **竞品诋毁/不正当竞争** | 竞品/水军捏造、虚假比较、贬低我方 | 走本流程 |

## 处置三步（合规反制，agent 不自行对线）
1. **取证固定**：第一时间截图/录屏/公证留存诋毁内容（时间、账号、链接），不删不回怼。
2. **走合规渠道反制**（由法务/公关主导，agent 只协助整理材料）：
   - 平台投诉/举报（抖音等平台的不正当竞争/诋毁举报通道）
   - 法务评估发律师函 / 必要时诉讼
   - 公关统一口径回应（如需）
3. **上报**：立即上报法务 + 公关 + 合规负责人，按高风险动作零决策处理。

## 🔴 agent 绝不做
- 不自行在评论区/公开平台和竞品对线、互撕
- 不自行发声明、不自行挂"律师函截图"（对外声明/法律动作属零决策权高风险动作，必须法务）
- 不捏造或夸大反击对方（我方反制也须合法，不得变成新的不正当竞争）

## agent 标准回应（老板问"要不要发声明/挂律师函"时）
"这属于竞品不正当竞争，建议：① 先取证固定（截图/公证）② 走平台投诉 + 法务评估发函，③ 对外声明须法务/公关定稿。我不能自行发声明或挂律师函截图，已帮你把材料整理好转法务。"

# 资质核验与医疗广告合规清单（医美核心）★

> 核心：**发布对外内容 / 上新项目前，先过这两道闸门。任一项勾不掉 → 🛑 停，转〔合规负责人〕，不得放行。**
> 法规以《医疗广告管理办法》《广告法》及主管部门要求为准（此处讲要点，具体条款以官方文本为准）。

## 一、资质核验清单（上新 / 上项目前，agent 逐项核对，缺一即拦）

| 核验项 | 要点 | 不达标动作 |
|---|---|---|
| ☐ 机构资质 | 〔机构名〕持有效《医疗机构执业许可证》，诊疗科目覆盖该项目 | 🛑 停，转〔合规负责人〕 |
| ☐ 医师资质 | 操作医师持《医师资格证》+《医师执业证》，且**执业范围**覆盖该项目 | 🛑 停，超范围零容忍 |
| ☐ 药械来源 | 注射类/光电类药械经正规渠道采购、有合法批准文号、可溯源 | 🛑 停，禁用来源不明药械 |
| ☐ 无超范围操作 | 项目不超出机构诊疗科目与医师执业范围 | 🛑 超范围零容忍，绝不放行 |

> 证照原件 = ⛔禁止入库（证件类，留原地索引）；agent 只读核验结论，不留存证件影像。

## 二、医疗广告合规自检清单（对外内容发布前，逐条勾选，任一勾不掉 → 改稿或拦截）

- ☐ 无**疗效保证**（不写"保证有效/必定见效/无效退款"等承诺）
- ☐ 无**治愈率 / 有效率 / 成功率**等数字承诺
- ☐ 无**绝对化用语**（最/第一/顶级/100%/唯一/根治）
- ☐ 无**前后对比图**诱导（术前 vs 术后对比图禁用）
- ☐ 不**利用患者形象 / 名义 / 评价作证明**（禁患者现身说法、禁好评截图当广告）
- ☐ 已**送医疗广告审查**并取得审查通过（医疗广告须先审后发）
- ☐ 严格区分"**医疗美容**"与"**生活美容**"，不把医疗项目包装成生活美容规避监管
- ☐ 无夸大、无误导、无与同行作贬低性比较

### 红线表（命中即拦截，无例外）
| 红线 | 典型违规话术（示例，禁用） |
|---|---|
| 疗效保证 | "包好""无效退款""一次见效永久不反弹" |
| 数字承诺 | "有效率 99%""治愈率行业第一" |
| 绝对化 | "全市最好""第一品牌""100% 安全无风险" |
| 前后对比 | 配"术前→术后"对比图诱导下单 |
| 患者证言 | "顾客实拍好评""真人现身说法" |

## 用法
agent 发布对外内容 / 上新项目前，先跑「资质核验」再跑「广告合规自检」；两道全过 + 经〔合规负责人〕审查留痕 → 才可放行。任何不确定 → 默认拦截上报。隐私数据处理与纠纷应对见 [[客户隐私保护与纠纷分级处理]]。

## 🎯 目标(Goal)
把一个在线咨询线索,从首次触达推进到到店面诊并成交,或记录明确流失原因,全程守住"线上不诊断/不报价/不承诺疗效"的合规边界。

## 📥 上下文(Context) —— 动手前必读全

## 🪜 指令(Instructions)
1. 黄金窗口内首回(示意 ≤ 3 分钟),开放式开场(禁"在吗"/二选一)
2. 挖需求:关注部位 / 预算区间 / 核心顾虑 → 初判等级
3. 线上只科普共性知识,**不诊断、不报价、不承诺效果**
4. 用邀约钩子把对话导向到店面诊(免费面诊/医师亲诊/限时名额示意)
5. 到店:咨询师承接 → 医师面诊判断 → 出个性化方案 → 报价成交
6. 异议按结构应对(给真实预期、强调资质与安全,不承诺效果)
7. 成交或记录流失原因 → 回流自我进化接口

## 🛠 工具(Tools)

## 🔒 权限(Permission)
- 可读:咨询SOP、分级、异议库、对外科普话术
- 🔴 受限:客户聊天原文/手机号/微信号/术前照 → 医疗隐私,永不外发、输出必脱敏打码
- 线上禁止:诊断、报价、疗效保证、绝对化用语(如"一定有效""零风险")
- 须人工确认:超标准报价优惠、涉及项目的医师资质追溯、未成年人医美(须监护人同意方可推进)

## ✅ 验收(Acceptance)
- 首回 ≤ 黄金窗口 ｜ 开场开放式 ｜ 线上无诊断/无报价/无疗效承诺 ｜ 邀约到店有钩子

- 详见 evals.md

# 咨询面诊转化 SOP · 人可读总览

> 给人看的全流程。Agent 执行请走 [`_task-card.md`](_task-card.md)。
> 医美核心:线上是"挖需求 + 建信任 + 邀到店",诊断与定价一律到店面诊完成。

## 一句话流程
黄金窗口温度开场 → 挖需求(部位/预算/顾虑)初判级 → 线上只科普不诊断不报价 → 邀约到店面诊 → 咨询师承接+医师面诊出方案 → 报价处理异议 → 成交/记录流失。

## 几条铁律
1. **线上三不**:不诊断、不报价、不承诺效果——这是医美合规红线,也是引导到店的钩子。
2. **黄金窗口首回**:首回越慢,流失越多;开场禁"在吗"、禁二选一,用开放式问句挖需求。
3. **报价只在面诊后**:医美按个性化方案定价,线上轻易报价=失客+合规风险。
4. **隐私红线**:客户对话/手机/微信/术前照 🔴 受限,绝不外发、输出必打码。
5. **资质可追溯**:涉及项目须可追溯医师资质与药械合规,不确定→人工确认。
6. **未成年须监护人**:未成年人医美须监护人到场同意,不得绕过。

## 判断题
1. 客户开口问"做这个项目多少钱" → 怎么处理?（答:线上不直接报价,说明医美按个性化方案定价,邀约到店面诊由医师评估后出方案报价）
2. 客户只问价、反复比价、不谈需求 → 判什么级?怎么打?（答:比价型,不卷价格,转价值/资质/安全锚点,争取到店面诊体验）

## 动作题
4. 在线咨询首回的黄金动作是什么?要挖哪三类信息?（答:黄金窗口内开放式开场;挖关注部位 / 预算区间 / 核心顾虑,据此初判级）

# 在线咨询到到店转化 SOP（脱敏示范）

> 核心:**线上不诊断、不报价、不承诺效果——线上的任务是挖需求、建信任、邀到店;诊断与定价一律到店面诊完成。**
> (响应时长/名额数为示意,客户填自己的真实设置)

## 阶段化清单

### 阶段一 · 黄金响应(线上)
- [ ] 黄金窗口内首回(示意 ≤ 3 分钟),开放式开场,禁"在吗"、禁二选一
- [ ] 30 秒看来源/资料初判等级(详见 [[客户分级与异议处理]])

### 阶段二 · 首咨挖需求(线上)
- [ ] 开放式提问挖三件事:**关注部位 / 预算区间 / 核心顾虑**
- [ ] 只科普共性知识,**不对个体下诊断**(如"你这个适合做XX")
- [ ] 客户问价 → 不报价,说明医美按个性化方案定价,价格需面诊后定

### 阶段三 · 合规边界(贯穿线上全程)⚠️
- [ ] **不诊断**:不替医师判断适应症/方案
- [ ] **不报价**:不在线上给具体项目价格/优惠承诺
- [ ] **不承诺效果**:禁"一定有效/保证变美/零风险/永久"等疗效与绝对化用语
- [ ] 客户聊天/手机/微信/术前照 = 🔴 医疗隐私,永不外发、输出必打码

### 阶段四 · 邀约到店(线上→线下)
- [ ] 给钩子:免费面诊 / 医师亲诊 / 限时面诊名额(示意)
- [ ] 话术结构:**共情顾虑 → 给到店独有价值 → 给具体时间二选一约**
- [ ] 约定到店时间 → 进 〔到店预约系统〕,提前提醒

### 阶段五 · 到店面诊(线下)
- [ ] **咨询师**承接接待、了解诉求、建立信任
- [ ] **医师**面诊判断适应症、给个性化方案(此处才出诊断)
- [ ] 涉及项目须可追溯医师资质与药械合规,不确定→人工确认
- [ ] 未成年人医美须监护人到场同意

### 阶段六 · 出方案与成交(线下)
- [ ] 咨询师据医师方案出报价,逐一处理异议(见 [[客户分级与异议处理]])
- [ ] 成交 / 记录流失原因 → 回流自我进化接口

# 客户分级与异议处理（脱敏示范）

> 核心:**先判级再打法;所有异议应对都守住"不承诺效果、给真实预期、强调资质与安全"。**
> (打法为示意,客户按真实话术库补充)

## 一、客户分级与打法

| 等级 | 判级标准 | 对应打法 |
|---|---|---|
| 高意向 | 部位明确、预算清晰、主动问到店/时间 | 快速邀约到店面诊,医师亲诊出方案,趁热成交 |
| 比价型 | 反复问价、只比价格、不谈需求 | **不卷价格**,转价值/资质/安全锚点,邀到店体验差异 |
| 犹豫型 | 有需求但反复纠结、回复慢 | 给真实预期降焦虑,限时面诊钩子推一把,转长期触达 |
| 纠结安全型 | 高度关注疼痛/副作用/风险/真假 | 重点讲资质+安全+真实案例(脱敏),邀面诊当面释疑 |
| 咨询白嫖型 | 只要免费攻略、无到店意愿 | 给共性科普即可,不投入深聊,标记低优先 |

## 二、高频异议应对结构

| 异议 | 标准应对结构(不承诺效果) |
|---|---|
| 怕疼 | 共情 → 说明麻醉/舒适化手段 → 给真实痛感预期 → 邀面诊体验 |
| 怕假/不自然 | 强调医师审美与资质 → 给真实自然预期(非"绝对自然")→ 看脱敏案例 |
| 怕副作用 | **不说"零风险"** → 说明正规药械与风险告知 → 强调资质保障 → 邀面诊评估 |
| 比价 | 不贬低同行 → 锚定资质/医师/药械正品/术后保障 → 邀到店感受差异 |
| 老公/家人不同意 | 共情 → 提供可对家人解释的客观信息 → 不施压 → 留长期触达 |

## 合规提醒 ⚠️
- 任何应对都**禁疗效承诺、禁夸大效果、禁绝对化**(如"一定有效""零风险""永久")
- 引用案例须脱敏,术前照属 🔴 医疗隐私不得外发
- 涉及适应症判断 → 一律导向医师面诊,不在线上诊断

# 未成年咨询处置流程（操作 SOP · 三步缺即停）

> 补"须监护人同意"红线的盲区：原则在，缺"家长缺位/不知情怎么办、是否书面、年龄怎么核"的操作动作。
> 核心：未成年医美**不是不能做，而是必须监护人到场书面同意**；任何"绕过家长尽快约"的诉求一律拒绝。

## 三步流程（缺任一即停，不得推进成交）
1. **核龄**：咨询中识别疑似未成年（学生/年龄信号）→ 主动、礼貌确认真实年龄；不得为成交装看不见。未满 18 岁 = 进入本流程。
2. **须监护人到场**：明确告知"未成年医美项目须监护人陪同到场、共同知情同意后方可评估"。线上只做科普答疑，**不报价、不诊断、不预约手术档**，可登记意向但状态标"待监护人确认"。
3. **书面同意留痕**：监护人到场后，须签署知情同意（书面留痕，参照隐私/知情铁律），核验监护关系。无书面 = 不推进。

## 家长缺位/不知情场景标准话术（agent 用）
- 客户说"家长不会同意/不想让家长知道" → 明确而温和拒绝："这类项目按规定必须监护人陪同同意，我们不能绕过哦。你可以先了解信息，等和家长沟通好、一起过来，我们再给专业评估。"
- 客户催"能不能先排上/尽快约" → "登记可以，但要监护人到场确认后才能正式安排，这是对你安全负责。"
- **绝不**：为"尽快/坚决"绕过监护人、不暗示"瞒着家长也能做"。

## 红线
- 🔴 未成年无监护人到场同意 → 不评估、不报价、不成交
- 🔴 不诱导未成年绕过监护人

# 复购与升单 SOP · 人可读总览

> 给人看的全流程。Agent 执行请走 [`_task-card.md`](_task-card.md)。

## 一句话流程
读已做项目定周期 → 复购窗口按节奏提醒 → 判升单时机走升单路径 → 关联升单只基于真实需求 → 高价值客户升会员 → 触发合规老带新。

## 几条铁律
1. **复购按项目真实周期**：光电/注射类有天然疗程与维持周期,按周期提醒,不是催单。
2. **升单不诱导过度医疗**：升单（单项目→疗程包→联合方案→年卡）必须基于真实医学需求 + 客户知情自愿。
3. **关联升单要可解释**：做了 A 推荐 B,须说得清医学逻辑,不确定 → 转医师确认。
4. **绝不承诺疗效**：任何复购/升单话术不得宣称疗效保证、不得绝对化用语。
5. **老带新物料合规**：转介绍禁用前后对比图诱导、禁疗效话术,双向激励用权益不用夸大效果。
6. **隐私红线**：会员消费数据/已做项目/手机微信 🔴 受限,绝不外发。

# 会员体系与老带新机制（脱敏示范）

> 核心：**用会员分层锁定长期复购,用合规老带新撬动转介绍——激励靠权益,绝不靠夸大效果。**
> （权益/礼品为示意,客户填自己机构的真实设置）

## 一、会员分层（示意）

- **〔储值会员〕**：储值享〔权益占位〕,适合高频护理客户
- **〔疗程会员〕**：锁定疗程包,享〔疗程价/优先排期〕
- **〔年卡会员〕**：长期绑定,享〔年度权益/专属顾问〕

> 权益设计原则：用确定性权益（价格/排期/服务）吸引,**不得以疗效承诺作为权益卖点**。

## 二、复购激励
- 到维持周期的会员 → 专属复购权益（衔接 [[项目周期复购与升单地图]]）
- 升级会员 = 升单路径终点,稳定客户优先引导

- **推荐礼设计**：〔推荐礼占位〕,双向激励——老客户得〔权益〕,新客户得〔权益〕
- **触发时机**：满意度高、消费稳定的老客户
- **物料合规（红线）**：
  - 🔴 禁用前后对比图诱导
  - 🔴 禁疗效话术、禁绝对化用语
  - 物料须经合规审核后再用,激励用权益表达

## 四、数据隐私（红线）
- 🔴 会员消费数据 / 已做项目记录 / 手机微信 = 医疗隐私,受限,绝不外发、绝不进对外物料,输出必脱敏。

# 到店钩子设计与转化路径（脱敏示范）

> 核心：**钩子是请客户进门的理由，不是免费午餐。** 引流款必须自带"防白嫖门槛 + 接面诊路径"，否则只引来薅羊毛的人。
> （价格/项目名为占位，客户填自己的真实设置）

## 钩子三类（按目的选）
| 钩子类型 | 示例 | 目的 | 防白嫖设计 |
|---|---|---|---|
| 引流款（低价体验）| 〔皮肤检测〕/〔小气泡体验价〕 | 拉到店 | 限新客、到店才享、需预约面诊 |
| 爆款团购券 | 〔爆款项目名〕团购套餐 | 放量转化 | 到店核销、绑定升单方案 |
| 到店礼 | 到店即送〔到店礼〕 | 提高到店率 | 仅核销当天领、不可转赠 |

## 防"只引白嫖党"四招
1. **设门槛**：引流款限新客、限量、需预约，不做人人可领的纯薅羊毛入口。
2. **绑面诊**：低价项目必须配一次到店面诊（皮肤检测/方案咨询），让客户和医师真实接触。
3. **价值锚点**：体验款讲清"原价 vs 体验价"，凸显这是体验机会不是廉价处理。
4. **升单路径**：每个引流款都预设"体验 → 面诊 → 正价项目"的衔接动线，引来就有下一步。

## 转化路径（公域钩子 → 到店 → 面诊）
```
公域钩子(引流款/团购/到店礼)
        ↓  预约/核销（埋追踪标识）
到店核销 → 一次到店面诊（皮肤检测/方案咨询）
        ↓  留资即转
交接 [[在线咨询到到店转化SOP]]（面诊转化工作流）
```

## 合规与隐私（必须遵守）
- 钩子文案禁疗效保证、禁绝对化用语、禁用前后对比图诱导；涉及项目须可追溯医师资质。
- 客户到店产生的术前照片/检测记录/病历 = 🔴 医疗隐私受限，绝不外发、输出必打码。

## 用法
先按目的选钩子类型，再套防白嫖四招，最后确认升单路径接得上面诊。到店客户 → 交 [[在线咨询到到店转化SOP]]。

# 医美三平台种草矩阵（脱敏示范）

> 核心：**三平台不是重复发同一条，是漏斗的三段分工。** 小红书拦截搜索、抖音放量、大众点评临门一脚。
> （价格/项目名为占位，客户填自己的真实设置）

| 平台 | 漏斗定位 | 主打内容形态 | 引流终点 |
|---|---|---|---|
| 小红书 | 口碑种草 + 搜索拦截 | 真人变美日记/项目科普/避坑攻略/探店笔记 | 私域（评论/主页钩子）|
| 抖音 | 泛流量 + 本地生活团购 | 短视频科普/医师出镜/团购挂车 | 团购核销 → 到店 |
| 大众点评 | 到店决策临门一脚 | 机构主页/真实评价/星级/团购套餐 | 直接到店核销 |

## 各平台打法清单
- **小红书**：铺"项目名+城市/机构"长尾关键词，拦截主动搜索的高意向人群；笔记结尾引私域（"主页领〔皮肤检测〕名额"），靠口碑量堆信任。
- **抖音**：医师/真人出镜做科普涨泛流量，挂〔团购体验价〕做本地生活转化；直播间承接，团购券导向到店。
- **大众点评**：维护机构主页评分与真实评价（决策最后一公里看这里），上架〔团购套餐〕，让搜索到的人直接下单到店。

## 三平台共同合规底线（必须遵守）
1. **禁前后对比图诱导**：不得用术前术后对比图作引流卖点（违规 + 涉医疗隐私）。
2. **禁疗效/绝对化话术**：不写"一次见效""包好""永久""最佳"等，不承诺疗效。
3. **必标资质**：涉及医疗项目须标注医疗机构资质，项目可追溯医师资质，不确定→人工确认。

## 用法
先定这次主攻哪一层平台，再选对应内容形态。三平台引来的留资/到店客户 → 统一交 [[到店钩子设计与转化路径]] 衔接面诊。

# 项目交付术后管理 SOP · 人可读总览

> 给人看的全流程。Agent 执行请走 [`_task-card.md`](_task-card.md)。

## 一句话流程
术前核查四项（缺一就停）→ 拍照归隐私区 → 医师执业范围内操作+批号可追溯 → 术后即时医嘱+留观 → 按 24h/3天/7天/1月回访 → 反应分级处理，严重并发症立即升级人工+医师。

## 几条铁律
1. **两条准入底线**：操作必须在持证医师执业范围内、术前知情同意书必签——缺一不开做。
2. **项目匹配先于操作**：客户诉求与项目适应证不符，由医师面诊决定，不硬做。
3. **批号可追溯**：用药用械批号逐项登记，无追溯不操作。
4. **隐私红线**：术前术后照片/病历/诊疗记录 🔴 受限，永不外发，输出必打码。
5. **严重反应不自判**：感染/栓塞等严重并发症 = 🔴 立即人工+医师介入+按规上报，agent 绝不自行处理。

# 客户主动推送隐私数据处理流（操作 SOP）

> 补隐私铁律的盲区：铁律讲"机构内部产生→存受限区→对外脱敏"，本流程专管**客户从微信等外部主动发来病历/照片**这一最高频入口。
> 核心：客户主动发来 ≠ 可以随便处理。一旦进入机构，立即按 🔴 受限医疗隐私管。

## 标准动作流（agent 收到客户主动推送的照片/病历时）
1. **即时定性**：客户发来的术后照/病历/诊疗记录 = 🔴 受限医疗隐私，从落地那一刻起按受限管。
2. **不在线诊断**：agent 绝不判断"正常/不正常"（医学判断须医师），先安抚 + 收集客观信息（部位/时间/症状）。
3. **急症先行**：若描述含 🔴 表现（剧痛/发白/花斑/化脓/视力异常等）→ 立即走 [[术后跟踪与不良反应处理]] 急症升级，不等流转。
4. **合规入库**：照片/记录存入该客户 `_受限/` 档案专属分区，**不落入草稿区、不进任何对外/共享目录**。
5. **转医师走定向通道**：需医师看 → 走**一对一定向通道**（指定医师私发/专用病历系统），**🔴 严禁丢进多人医师群/工作群/对外群**（群转发=超授权扩散，最高发隐私事故）。转发时只给必要的医师，附最小必要信息（最小授权原则）。
6. **留痕**：登记"收到时间/客户/处理人/转给谁"，便于追溯。

## 红线（agent 绝不做）
- 🔴 把客户照片/病历转发到任何群（含内部医师群/工作群）
- 🔴 在线上对诊疗结果下判断
- 🔴 把受限照片用于案例/对外（即使脱敏也须另走授权流程）

## 不确定时
默认按受限处理 + 转 DRI/医师，宁可保守不可扩散。

# 术后跟踪与不良反应处理

> 核心：**轻微反应可按模板安抚+提醒，严重并发症 = 🔴 必须人工+医师介入并按规上报，agent 不得自行判断处理。**
> （回访话术/恢复期模板客户填自己机构的真实版本）

## 回访节奏（按项目分项跟踪）
| 时间点 | 重点 | 动作 |
|---|---|---|
| 术后 24h | 即时反应、红肿渗血 | 主动回访，确认无急性异常 |
| 术后 3 天 | 恢复期高峰、肿胀淤青 | 按〔恢复期模板〕提醒护理 |
| 术后 7 天 | 拆线/消肿/初步效果 | 确认恢复轨迹，约复诊 |
| 术后 1 月 | 最终效果、满意度 | 评估、记录、按需安排复查 |

## 不良反应分级与处理
| 分级 | 表现（示意） | 处理 |
|---|---|---|
| 🟢 轻微 | 轻度红肿、淤青、短暂胀痛 | 按〔恢复期模板〕安抚+护理提醒，记录观察 |
| 🟡 需关注 | 持续红肿、硬结、低热 | 提醒就诊复查，通知医师评估，记录 |
| 🔴 严重 | 感染、剧痛、皮肤发白/花斑（疑栓塞）、视力异常、过敏休克 | **立即指导就医/转诊 → 人工+医师介入 → 按规上报**，agent 不得自行判断处理 |

## 何时必须立即就医/转诊
- 出现 🔴 任一表现 → 第一时间指导客户立即就医，同步医师，不得拖延。
- 疑似血管栓塞（皮肤发白/剧痛/花斑）属急症 → 争分夺秒，立即人工介入。

## 红线
- 严重不良反应 = 🔴 必须人工 + 医师介入 + 按不良反应上报流程上报，**agent 绝不自行诊断或处理**。
- 术后照片/病历 = 🔴 医疗隐私，受限永不外发，输出必打码。

# 项目交付标准流程（术前-术中-术后）

> 核心：**操作必须在持证医师执业范围内、术前知情同意书必签——两条缺一不可，否则不得开始。**
> （以下为标准动作清单，客户填自己机构的真实模板与项目名）

## 术前 · 准入核查（缺一项就停）
- [ ] 核对操作者资质：项目须落在**持证医师执业范围内**，超范围 → 🔴 停止、转诊
- [ ] 核对项目匹配：客户诉求 ↔ 〔项目名〕适应证一致；**不匹配不开做**，由医师面诊决定
- [ ] 过敏史 + 禁忌症核查：逐项问诊填表，命中禁忌 → 🔴 停止
- [ ] 签署〔知情同意书〕：风险/恢复期/费用讲清，**客户本人签字**，未签不得开始
- [ ] 术前拍照存档：照片即时归入 🔴 医疗隐私受限区，永不外发、输出必打码

## 术中 · 操作执行
- [ ] 仅由持证医师在执业范围内操作，agent/助理不得替代医疗判断
- [ ] 三查七对意识：查客户/查项目/查部位，对身份对药械
- [ ] 记录用药用械**批号可追溯**：〔药械批号登记表〕逐项录入
- [ ] 全程记录写入〔诊疗记录〕，归医疗隐私受限区

## 术后 · 即时交付
- [ ] 即时医嘱：按〔医嘱模板〕口头+书面双交付
- [ ] 注意事项交付：恢复期禁忌、用药、复诊时间写清
- [ ] 留观：按项目规定时长观察，无异常方可离开
- [ ] 移交跟踪：客户信息进 [[术后跟踪与不良反应处理]] 的回访节奏

## 红线
- 操作超执业范围 / 知情同意未签 / 项目不匹配 → 🔴 一律停止，人工+医师介入。
- 术前术后照片、诊疗记录 = 🔴 医疗隐私，受限永不外发，任何输出必打码。

# 权限分级 · 四级

> 企业库和个人库最硬的区别。没有权限治理 = AI 资料垃圾场 + 隐私事故。

| 级别 | 含义 | 典型资产 | agent 行为 |
|---|---|---|---|
| 🟢 公开 | 可对外发布 | 对外物料、方法论、介绍 | 可写入对外产物 |
| 🟡 内部 | 团队内用 | 话术、SOP、运营数据 | 内部可用,不对外 |
| 🔴 受限 | 隐私/敏感 | 客户对话、通话记录、手机号、名单 | **绝不写入对外产物,只内部分析** |
| ⛔ 禁止入库 | 不该进库 | 证件、银行信息、未授权第三方资料 | 拒绝入库 |

## agent 红线
1. 🔴 受限资产 → 输出时必须脱敏/打码,绝不原文外发
2. 超标准的报价/优惠/对外承诺 → 必须人工确认
3. 不确定权限 → 默认按更高一级处理,问用户

## 医美红线（agent 必须遵守）
1. **医疗广告合规**：对外内容不得宣称疗效保证、不得用前后对比图诱导、不得绝对化用语。
2. **诊疗隐私**：术前术后照片/病历属医疗隐私，🔴 受限，绝不进定版、绝不对外、输出必打码。
3. **资质追溯**：涉及项目须可追溯医师资质与药械合规，不确定→人工确认。
4. **纠纷处理**：客诉/纠纷走 `compliance-risk` 工作流，高风险动作（赔付/对外声明）必须人工把关。

> 合规与风控是医美独立工作流 `compliance-risk`，不是附属——这是医美区别于其他行业的核心。
